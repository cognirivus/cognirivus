# Navigation Guide

Cognirivus features a multi-pane layout designed to put maximum information and control at your fingertips while keeping the interface clean and responsive.

---

## Layout Structure

On desktop, the application is divided into three key columns:

1. **Left Sidebar**: Your main control panel. It houses core navigation, communities you are a member of, and your user profile settings. You can collapse it into an icon-only strip using the toggle button or the `Ctrl+B` (or `Cmd+B`) keyboard shortcut.
2. **Main Content Area**: Where feeds, posts, collections, and chat panels render.
3. **Right Aside Panel**: Contextual information. For feed pages, this displays the **Similar Links Sidebar** (helping you discover related reading). For community chat, it lists active members.

---

## Sidebar Sections

- **Feed**: Quick access to your main dashboard.
- **Sources**: Add, organize, and monitor websites and feeds you follow.
- **Collections**: View and manage your personal and followed collections.
- **Communities**: Browse the community list or jump into community-specific pages.
- **Chat**: Access direct messages and private conversations.
- **Submit**: Create a new post (text, link, or media) to the network.

---

## Keyboard Shortcuts

Increase your productivity with these global keyboard shortcuts:

| Shortcut         | Action                               |
| ---------------- | ------------------------------------ |
| `⌘ K` / `Ctrl+K` | Open Search / Command Palette        |
| `⌘ B` / `Ctrl+B` | Toggle Left Sidebar                  |
| `g` then `f`     | Go to Feed                           |
| `g` then `s`     | Go to Sources                        |
| `g` then `c`     | Go to Collections                    |
| `g` then `m`     | Go to Communities                    |
| `n`              | Create a New Post                    |
| `Esc`            | Close dialogs, dropdowns, and search |

---

## Responsive Design

- **On Mobile**: The left sidebar collapses into a slide-out drawer (accessible via the hamburger menu in the top header). The right aside panel collapses into an overlay panel. A convenient bottom tab bar appears for quick navigation on phone screens.
